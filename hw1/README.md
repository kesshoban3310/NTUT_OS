### problem 2.24

> [!TIP]  
> 檔名為 `FileCopy.cpp`。  
> 編譯方式為 `gcc FileCopy.cpp -o FileCopy`。  
> `./FileCopy <src> <dsc>` 為使用方法，像是 `./FileCopy input.txt output.txt` 。  
> 檢查參數正確、是否有來源檔案，目的檔案名稱是否重複等功能。  
> 使用 `strace` 之檢查檔案為 `trace_log.txt` 。

> [!TIP]
